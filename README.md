# LOOGIE — Avatar Store

A fictional avatar storefront with a slime/snot/loogie aesthetic. Browse avatars, add them to your cart, and pretend you can actually buy them.

Built with **Vite + React + TypeScript + Tailwind CSS**.

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

Push to `main` — GitHub Actions builds and deploys to GitHub Pages automatically.
